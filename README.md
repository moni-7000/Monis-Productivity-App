# Moni's Productivity App 
*by Monica Ago*

A simple pink/dark neon productivity app.

## Features
- 4 Function calculator
- Daily todo-list with data persistence 
- Focus timer with alarm
- Light and Dark mode themes

## Tools
- Python
- Custom Tkinter
- Sqlite3
- Pygame

## Installation
Simply install the executable and run to use!
*Windows defender will likely flag this because it was made via pyinstaller and not digitally signed. Though, it is a false-positive.*
